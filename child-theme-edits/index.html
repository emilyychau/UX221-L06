Child Theme Edits

First, I uploaded the One Click Child Theme plugin into my playground.wordpress workspace. I chose The original, parent theme used for this assignment is called Variations. Then, I renamed the child theme to “UX221-L06”.

To change the style.css of the newly made child theme, I went to the Dashboard > Tools > Theme File Editor.

Fonts

To further customize this child theme, I added the @import statement from Google Fonts and I set the 2nd level header (H2) to Poppins.

Colour

The colour of the header and footer were set to an emerald green colour. for increased contrast and legibility, the body text was set to an off-white colour instead of the default black colour text.
